BSBE Renewed


A shader for Minecraft: Bedrock Edition that aims to be beautiful with colorful tones and some features like underwater caustic, 2D cloud, water reflection, water bump, etc.




Changelog

  • minor improvements and optimizations
  • added support for Minecraft with the Renderdragon Engine


Note

  • This shader is only supported for Minecraft: Bedrock Edition on Android and iOS
  • The Renderdragon material is only for MCBE Android version
  • Don't reupload the shader file. If you want to share it, just use this github page link
  • Don't make this github page or download link as a shortlink with some payment method activated on that.
